# Python-U.S_States_Guessing_Game

An implementation of an interactive U.S States guessing game using Python Turtle Graphics

Beginning Screen:

[Start Screen](https://i.postimg.cc/jS9gpRsV/GH12.png)

As you start the game a pop up appears where user can enter thier guess and if its right the state name will be diplayed on map and the cycle continues until all the 50 U.S states are guessed.

Finishing Screen:

[Final_Screen](https://i.postimg.cc/CxtP3wFT/GH11.png)

Once the user completes guessing all the 50 states the game automoatically ends.

User Termination:

If incase the player wants to give up and end the game in-between he can enter the word 'Exit' to terminate the game manually

[Termination Input](https://i.postimg.cc/xd8JBbfr/GH13.png)

Once the user enters exit and clock Ok. Then the games manually terminates with a terminiation message.

[Termination_Screen](https://i.postimg.cc/bv5DPNrs/GH14.png)

As displayed in the termination message a CSV file will be created upon manual termination, which contains the list of states the user failed to guess.
